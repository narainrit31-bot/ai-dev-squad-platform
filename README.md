## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- PostgreSQL 14+
- Docker & Docker Compose

### Installation

```bash
# Clone repository
git clone https://github.com/narainrit31-bot/ai-dev-squad-platform.git
cd ai-dev-squad-platform

# Setup environment
cp .env.example .env

# Install dependencies
npm install

# Start with Docker
docker-compose up -d

# Run migrations
npm run db:migrate

# Start development
npm run dev
```
